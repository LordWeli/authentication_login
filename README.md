## <a name="about"></a>About
  System to perform user authentication with username and password.

## <a name="how-to-use"></a>How To Use
  -  Clone the project
  -  Run the bundle install
      -  rails db:create
      -  rails db:migrate
      -  rails db:seed
      -  and be happy :grin: :heart:
